Bootloader & Kernel
============

Hiérarchie 
----------
`config` 
:     Contient le dernier .config à jour

`u-boot-sunxi` 
:     Contient un ensemble de fichiers permettant de générer le bootloader : fork du projet sunxi

`Makefile`  
:     Permet de générer : le bootloader, une image kernel et/ou les modules kernel
